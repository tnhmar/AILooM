## Objective

Instrument all engine hot paths with Prometheus counters and histograms.
Add a `/metrics` endpoint to the FastAPI app. All metrics must carry
`tenant_id` and `operation` labels, be defined in a single registry module,
and be opt-in via `ObservabilitySettings.metrics_enabled`.
