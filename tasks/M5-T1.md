## Objective

Implement the production-facing FastAPI application exposing the core memory use cases
as HTTP endpoints. This is the primary integration boundary for external services.
The server must be thin: request validation, tenant resolution, use-case dispatch,
response mapping, and error translation only.

**Generate `src/memory_layer/api/schemas.py`:**

Define Pydantic request/response schemas for:
- `WriteMemoryRequestModel`
- `WriteMemoryResponseModel`
- `SearchMemoryRequestModel`
- `SearchMemoryResponseModel`
- `RecallMemoryRequestModel`
- `RecallMemoryResponseModel`
- `ExplainRecallResponseModel`
- `ErrorResponseModel`

Rules:
- Schemas must be decoupled from domain dataclasses.
- Include explicit enum serialization as strings.
- Add `model_config = ConfigDict(extra="forbid")` on all request models.

---

**Generate `src/memory_layer/api/errors.py`:**

Implement FastAPI exception handlers mapping:
- `MemoryNotFoundError` → 404
- `ValidationError` / domain validation exception → 400
- `TenantIsolationError` → 403
- `CapabilityNotAvailableError` → 409
- generic `Exception` → 500

Each returns `ErrorResponseModel` with fields:
`error_code`, `message`, `details`, `trace_id`.

---

**Generate `src/memory_layer/api/app.py`:**

Implement:

```python
app = FastAPI(title="memory-layer", version="0.1.0")
```

Add dependency-injected endpoint handlers for:
1. `POST /v1/memories:write`
   - header `X-Tenant-Id`
   - calls `WriteMemoryUseCase.execute`
2. `POST /v1/memories:search`
   - calls `SearchMemoryUseCase.execute`
3. `POST /v1/memories:recall`
   - calls `RecallMemoryUseCase.execute`
4. `GET /v1/memories/{memory_id}`
   - calls `GetMemoryUseCase.execute`
5. `DELETE /v1/memories/{memory_id}`
   - query param `actor`
   - calls `DeleteMemoryUseCase.execute`
6. `GET /v1/traces/{trace_id}`
   - calls `ExplainRecallUseCase.execute`
7. `POST /v1/sessions/{session_id}:end`
   - body contains `scope`
   - calls `NotifySessionEndedUseCase.execute`
8. `POST /v1/admin/tenants/{tenant_id}:decay`
   - calls `DecayUseCase.execute`
9. `POST /v1/admin/tenants/{tenant_id}:consolidate`
   - calls `ConsolidateUseCase.execute`

Rules:
- Create explicit dependency provider functions like `get_write_use_case()`.
- Convert API models → domain objects explicitly.
- Convert domain objects → API models explicitly.
- Read tenant from `X-Tenant-Id` for user endpoints; admin endpoints take path tenant_id.
- Do not instantiate real services; leave dependency providers as stubs raising
  `NotImplementedError` so tests can override them.
- Register all exception handlers from `errors.py`.
- Add `/healthz` returning `{"status": "ok"}`.
