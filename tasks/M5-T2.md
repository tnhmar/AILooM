## Objective

Implement the first-party Python SDK for consuming the HTTP API. The SDK should
provide a clean, typed, ergonomic wrapper over the REST endpoints and hide
HTTP/JSON boilerplate from application code.

**Generate `src/memory_layer/sdk/client.py`:**

Implement `MemoryLayerClient` with:

```python
class MemoryLayerClient:
    def __init__(self, base_url: str, tenant_id: str, timeout: float = 10.0) -> None: ...
    async def aclose(self) -> None: ...

    async def write(self, request: SDKWriteRequest) -> SDKWriteResponse: ...
    async def search(self, request: SDKSearchRequest) -> SDKSearchResponse: ...
    async def recall(self, request: SDKRecallRequest) -> SDKRecallResponse: ...
    async def get_memory(self, memory_id: str) -> dict: ...
    async def delete_memory(self, memory_id: str, actor: str) -> None: ...
    async def explain(self, trace_id: str) -> SDKMemoryTrace: ...
    async def end_session(self, session_id: str, scope: dict) -> None: ...
    async def healthz(self) -> dict: ...
```

Rules:
- Internally use `httpx.AsyncClient`.
- Send `X-Tenant-Id` header automatically.
- On non-2xx, parse API error payload and raise `MemoryLayerHTTPError`.
- On transport-level failures (`httpx.HTTPError`), raise `MemoryLayerTransportError`.
- Convert JSON payloads into SDK models explicitly.
- Support `async with MemoryLayerClient(...) as client:`.
