## Objective

Implement a layered, validated configuration system that loads settings from
environment variables, optional TOML/YAML file, and safe defaults. All engine
components, adapters, and the API server must read from a single `Settings`
object injected at startup — no scattered `os.getenv` calls.

**Generate `src/memory_layer/config/settings.py`:**

Define a single `Settings` class with nested sub-models:

```python
class ServerSettings(BaseModel):
    host: str = "0.0.0.0"
    port: int = 8000
    workers: int = 1
    log_level: str = "INFO"
    reload: bool = False

class StorageSettings(BaseModel):
    backend: Literal["sqlite", "postgres"] = "sqlite"
    sqlite_path: str = "./data/memory_layer.db"
    postgres_dsn: Optional[str] = None
    vector_backend: Literal["chroma", "qdrant", "pgvector"] = "chroma"
    chroma_path: str = "./data/chroma"
    qdrant_url: Optional[str] = None
    qdrant_api_key: Optional[str] = None

class EmbeddingSettings(BaseModel):
    provider: Literal["openai", "ollama", "sentence-transformers"] = "openai"
    model_id: str = "text-embedding-3-small"
    dimensions: int = 1536
    api_key: Optional[str] = None
    base_url: Optional[str] = None

class LLMSettings(BaseModel):
    provider: Literal["openai", "anthropic", "ollama"] = "openai"
    model_id: str = "gpt-4o-mini"
    api_key: Optional[str] = None
    base_url: Optional[str] = None
    max_tokens: int = 2048
    temperature: float = 0.0

class SchedulerSettings(BaseModel):
    decay_interval_seconds: int = 3600
    consolidation_interval_seconds: int = 1800
    enabled: bool = True

class ObservabilitySettings(BaseModel):
    metrics_enabled: bool = True
    tracing_enabled: bool = False
    otlp_endpoint: Optional[str] = None
    service_name: str = "memory-layer"
    log_json: bool = False

class Settings(BaseSettings):
    server: ServerSettings = ServerSettings()
    storage: StorageSettings = StorageSettings()
    embedding: EmbeddingSettings = EmbeddingSettings()
    llm: LLMSettings = LLMSettings()
    scheduler: SchedulerSettings = SchedulerSettings()
    observability: ObservabilitySettings = ObservabilitySettings()

    model_config = SettingsConfigDict(
        env_prefix="MEMORY_LAYER_",
        env_nested_delimiter="__",
        env_file=".env",
        env_file_encoding="utf-8",
        case_sensitive=False,
    )
```

**Generate `tests/config/test_settings.py`:**

Tests:
1. `Settings()` constructs with all defaults.
2. `server.port` default is `8000`.
3. `storage.backend` default is `"sqlite"`.
4. `embedding.provider` default is `"openai"`.
5. Env var `MEMORY_LAYER_SERVER__PORT=9090` overrides `server.port`.
6. Env var `MEMORY_LAYER_STORAGE__BACKEND=postgres` overrides `storage.backend`.
7. Nested env var `MEMORY_LAYER_LLM__MODEL_ID=gpt-4o` overrides `llm.model_id`.
8. `get_settings()` returns the same instance on two calls (singleton).
9. `override_settings(custom)` causes `get_settings()` to return `custom`.
10. `reset_settings()` clears singleton so next `get_settings()` reloads.
11. Invalid value for `storage.backend` raises `ValidationError`.
12. `ObservabilitySettings` default has `metrics_enabled=True` and `tracing_enabled=False`.
