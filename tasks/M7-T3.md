## Objective

Implement `PgvectorVectorIndex` — a concrete `VectorIndexPort` backed by the
pgvector Postgres extension. Colocates vectors with records in the same DB.
Uses HNSW index for approximate nearest-neighbor search. Supports tenant-scoped
hybrid metadata filtering.

```sql
CREATE EXTENSION IF NOT EXISTS vector;

CREATE TABLE IF NOT EXISTS vector_index (
    memory_id            TEXT        NOT NULL,
    tenant_id            TEXT        NOT NULL,
    embedding            vector(1536),
    embedding_model_id   TEXT        NOT NULL,
    embedding_dimensions INT         NOT NULL,
    content              TEXT        NOT NULL,
    sector               TEXT        NOT NULL,
    lifecycle_state      TEXT        NOT NULL,
    metadata             JSONB       NOT NULL DEFAULT '{}',
    PRIMARY KEY (memory_id, tenant_id)
);

-- HNSW index (tunable)
CREATE INDEX IF NOT EXISTS idx_vector_index_hnsw
    ON vector_index
    USING hnsw (embedding vector_cosine_ops)
    WITH (m = 16, ef_construction = 64);

CREATE INDEX IF NOT EXISTS idx_vector_index_tenant_sector
    ON vector_index (tenant_id, sector, lifecycle_state);
```

Implement `PgvectorVectorIndex`:

```python
class PgvectorVectorIndex:
    """
    Implements VectorIndexPort.

    upsert(doc):
      INSERT INTO vector_index (...) VALUES (...)
      ON CONFLICT (memory_id, tenant_id) DO UPDATE SET
        embedding=$3, content=$4, sector=$5, lifecycle_state=$6, metadata=$7.

    search(query_embedding, tenant_id, k, filters):
      SELECT memory_id, content, metadata,
             1 - (embedding <=> $1::vector) AS score
      FROM vector_index
      WHERE tenant_id=$2
        AND lifecycle_state = ANY($3)   -- default: [ACTIVE, CONSOLIDATED]
        AND sector = ANY($4)            -- from filters.get("sectors", all)
      ORDER BY embedding <=> $1::vector
      LIMIT $5;
      Returns List[VectorSearchResult].

    delete(memory_id, tenant_id):
      DELETE WHERE memory_id=$1 AND tenant_id=$2.
    """

    def __init__(self, pool: asyncpg.Pool, dimensions: int = 1536) -> None: ...

    async def ensure_schema(self) -> None:
        """Run VECTOR_INDEX_DDL against the pool. Call at startup."""

    async def upsert(self, doc: VectorDocument) -> None: ...
    async def search(
        self,
        query_embedding: List[float],
        tenant_id: TenantId,
        k: int,
        filters: dict,
    ) -> List[VectorSearchResult]: ...
    async def delete(self, memory_id: MemoryId, tenant_id: TenantId) -> None: ...
```

Rules:
- Register pgvector codec on each connection: `await pgvector.asyncpg.register_vector(conn)`.
- Cosine similarity score = `1 - (embedding <=> query)` (already in range [-1, 1]).
- `filters` dict may contain keys: `sectors` (list), `lifecycle_states` (list).
  Default lifecycle_states filter: `[ACTIVE, CONSOLIDATED]`.
- HNSW `ef_search` can be set per-query via `SET LOCAL hnsw.ef_search = 100`.
