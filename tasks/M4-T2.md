## Objective

Implement `ConsolidationService` — the concrete `ConsolidateUseCase` that groups
raw episodic/verbatim memory records into higher-level consolidated summaries.
This is the "sleep cycle" analogue: it merges redundant ACTIVE records, promotes
important ones to CONSOLIDATED, and prunes low-salience duplicates.

**Generate `src/memory_layer/engine/consolidation.py`:**

```python
class ConsolidationService:
    """
    Implements ConsolidateUseCase.

    Consolidation flow:
    1. Emit ConsolidationJobStartedEvent.
    2. Load ConsolidationPolicy via TenantPolicyRepositoryPort.
       If policy.enabled=False, emit ConsolidationJobCompletedEvent(records_processed=0)
       and return 0 immediately.
    3. Resolve target sectors: policy.sectors (all if empty).
    4. For each target sector:
       a. List ACTIVE records for that sector within scope
          (up to policy.max_items_per_run).
       b. If count < policy.threshold_record_count: skip this sector.
       c. Group records by (agent_id, user_id) within the scope.
       d. For each group of records:
          - If llm_client is provided: call _summarise(records) to produce a
            consolidated summary string.
          - Else: concatenate raw_payload with newline separator (no-LLM fallback).
          - Write new MemoryRecord with:
              sector=same sector, lifecycle_state=CONSOLIDATED,
              pipeline_status=ENRICHED (already processed),
              raw_payload=summary string,
              scope=group scope.
          - Update each source record lifecycle_state to CONSOLIDATED
            via record_repo.update_lifecycle.
          - Append AuditEntry(CONSOLIDATE, SUCCESS) per source record.
          - Emit MemoryConsolidatedEvent per source record.
    5. Emit ConsolidationJobCompletedEvent(records_processed=N, duration_ms=...).
    6. Return N (total source records consolidated).

    _summarise(records):
    - Build a prompt listing record payloads.
    - Call LLMClientPort.complete(system_prompt, user_prompt).
    - Return the LLM response string.
    - On any exception: log warning, return newline-joined fallback.
    """

    def __init__(
        self,
        record_repo: MemoryRecordRepositoryPort,
        audit_log: AuditLogPort,
        observer: ObserverPort,
        policy_repo: TenantPolicyRepositoryPort,
        llm_client: Optional[LLMClientPort] = None,
    ) -> None: ...

    async def execute(
        self, tenant_id: TenantId, scope: Optional[Scope] = None
    ) -> int: ...

    async def _summarise(self, records: List[MemoryRecord]) -> str: ...
```
