## Objective

Implement a LangGraph-native adapter layer so the memory framework can be plugged
into graph-based agents as reusable nodes/helpers rather than only as an external API.
This should make memory operations first-class graph steps.

---

## Perplexity Prompt — paste this verbatim

You are implementing the LangGraph integration adapter for `memory-layer`.
Files:
- `src/memory_layer/integrations/langgraph.py`
- `src/memory_layer/integrations/state_models.py`

The project targets production-grade agent systems using LangGraph patterns,
but this adapter must remain import-safe even if LangGraph is not installed.
Use stdlib typing and optional imports only.

---

**Generate `src/memory_layer/integrations/state_models.py`:**

Define lightweight dataclasses / TypedDicts for graph state fragments:
- `MemoryWriteState`
- `MemorySearchState`
- `MemoryRecallState`

Each should hold the minimum fields required to call the SDK / use cases.

---

**Generate `src/memory_layer/integrations/langgraph.py`:**

Implement:

```python
def build_write_memory_node(client: MemoryLayerClient): ...
def build_search_memory_node(client: MemoryLayerClient): ...
def build_recall_memory_node(client: MemoryLayerClient): ...
```

Rules:
- Each builder returns an async callable `(state) -> dict` suitable for use as a LangGraph node.
- The node reads the relevant state fragment, calls the SDK client, and returns
  a partial state update dict.
- Keep LangGraph itself optional: do not import `langgraph` at module import time.
- Validate required keys and raise `ValueError` for malformed state.
- Return keys:
  - write node → `{ "memory_write_result": ... }`
  - search node → `{ "memory_search_result": ... }`
  - recall node → `{ "memory_recall_result": ... }`
- Add module-level helper docs showing expected state shape.
```

---

**Generate `tests/integrations/test_langgraph.py`:**

Use `pytest-asyncio`. Mock `MemoryLayerClient` methods with `AsyncMock`.

Tests:

4. Write node reads state and returns `memory_write_result` key.

8. Missing required state field raises `ValueError` in search node.

10. Module import does not require LangGraph to be installed.
