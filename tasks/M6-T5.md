## Perplexity Prompt — paste this verbatim

You are writing packaging and CI config files for `memory-layer`.
Files to generate:
- `Dockerfile`
- `docker-compose.yml`
- `.github/workflows/ci.yml` (update existing)
- `scripts/entrypoint.sh`

The project uses `uv` as package manager and Python 3.11+.
The FastAPI app is in `src/memory_layer/api/app.py` with factory `create_app()`.
The server is launched via `uvicorn`.

---

**Generate `Dockerfile`:**

```dockerfile
# Stage 1: builder
FROM python:3.11-slim AS builder
RUN pip install uv
WORKDIR /app
COPY pyproject.toml uv.lock ./
RUN uv sync --frozen --no-install-project
COPY src/ ./src/
RUN uv sync --frozen

# Stage 2: runtime
FROM python:3.11-slim AS runtime
WORKDIR /app
COPY --from=builder /app/.venv ./.venv
COPY --from=builder /app/src ./src
COPY scripts/entrypoint.sh ./entrypoint.sh
RUN chmod +x entrypoint.sh

ENV PATH="/app/.venv/bin:$PATH"
ENV PYTHONPATH="/app/src"
EXPOSE 8000

USER nobody
ENTRYPOINT ["./entrypoint.sh"]
```

Rules:
- No root user in runtime stage.
- Use `--frozen` to ensure reproducible builds from `uv.lock`.
- Image should be minimal: no dev deps in runtime stage.

**Generate `scripts/entrypoint.sh`:**

```bash
#!/bin/sh
set -e
exec uvicorn memory_layer.api.app:create_app \
  --factory \
  --host ${MEMORY_LAYER_SERVER__HOST:-0.0.0.0} \
  --port ${MEMORY_LAYER_SERVER__PORT:-8000} \
  --workers ${MEMORY_LAYER_SERVER__WORKERS:-1}
```

**Update `.github/workflows/ci.yml`** with these jobs:

```yaml
jobs:
  lint:
    - ruff check src/ tests/
    - ruff format --check src/ tests/

  typecheck:
    - mypy src/ --strict --ignore-missing-imports

  test:
    - pytest tests/ -m "not benchmark"
      --cov=src/memory_layer
      --cov-fail-under=90
      --cov-report=xml

  security:
    - pip-audit --requirement requirements.txt

  benchmark:
    if: github.event_name == 'push' && github.ref == 'refs/heads/main'
    - pytest tests/benchmarks/ -m benchmark -v
      --json-report --json-report-file=benchmark-results.json
    - Upload benchmark-results.json as artifact

  docker-build:
    - docker build -t memory-layer:ci .
    - docker run --rm memory-layer:ci /app/.venv/bin/python -c
      "from memory_layer.api.app import create_app; print('OK')"
```

Rules:
- Jobs run on `ubuntu-latest`, Python 3.11.
- `test` job must fail if coverage < 90%.
- `benchmark` job only runs on push to main (not on PRs).
- `security` job must fail if any known CVE is found.
