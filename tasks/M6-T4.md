**Generate `src/memory_layer/api/health.py`:**

```python
@dataclass
class ComponentHealth:
    name: str
    status: Literal["ok", "degraded", "down"]
    latency_ms: Optional[float] = None
    detail: Optional[str] = None

@dataclass
class HealthReport:
    status: Literal["ok", "degraded", "down"]
    version: str
    components: List[ComponentHealth] = field(default_factory=list)
    checked_at: datetime = field(default_factory=datetime.utcnow)

class HealthChecker:
    """
    Runs probe functions for each registered component.
    Overall status:
      - "ok" if all components ok
      - "degraded" if any degraded (but none down)
      - "down" if any component is down
    """
    def __init__(self, version: str = "0.1.0") -> None: ...

    def register(
        self,
        name: str,
        probe: Callable[[], Awaitable[ComponentHealth]],
    ) -> None: ...

    async def check(self) -> HealthReport: ...


async def probe_record_repo(repo: MemoryRecordRepositoryPort) -> ComponentHealth: ...
async def probe_vector_index(index: VectorIndexPort) -> ComponentHealth: ...
async def probe_full_text_index(index: FullTextIndexPort) -> ComponentHealth: ...
```

Update `api/app.py`:
- Replace basic `/healthz` with a deep health check: `GET /healthz` returns `HealthReport` JSON.
- Add `GET /readyz` that returns 200 only when all components are `"ok"`, else 503.

```python
# Mark as benchmark: @pytest.mark.benchmark
# Skip in normal CI with: pytest -m "not benchmark"

FIXTURES = [
    {
        "memory": "User's name is Alice and she lives in Montreal.",
        "query": "What is the user's name?",
        "expected_in_result": "Alice",
    },
    {
        "memory": "Preferred LLM is Claude 3.5 Sonnet for coding tasks.",
        "query": "Which LLM does the user prefer for coding?",
        "expected_in_result": "Claude",
    },
    # Add 3 more covering temporal, contradiction, and procedural scenarios
]

async def run_benchmark(client: MemoryLayerClient) -> BenchmarkResult:
    """
    For each fixture:
    1. Write memory.
    2. Recall with matching query.
    3. Check expected_in_result in any RecallItem.content.
    Returns BenchmarkResult(total, passed, recall_at_1, recall_at_5).
    """
```

Tests:
1. `probe_record_repo` returns `ComponentHealth` with `status="ok"` for working repo.
2. `probe_record_repo` returns `status="down"` when repo raises.
3. `probe_vector_index` returns `ComponentHealth` with `status="ok"` for working index.
4. `HealthChecker.check` returns `status="ok"` when all probes pass.
5. `HealthChecker.check` returns `status="down"` when any probe returns down.
6. `HealthChecker.check` returns `status="degraded"` when any probe is degraded.
7. `/healthz` returns 200 with JSON body containing `status` field.
8. `/readyz` returns 200 when all healthy.
9. `/readyz` returns 503 when any component is down.
