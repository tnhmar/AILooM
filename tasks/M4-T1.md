## Objective

Implement `DecayService` — the concrete `DecayUseCase` that enforces `RetentionPolicy`
by scanning for ACTIVE/CONSOLIDATED records whose age exceeds policy thresholds and
transitioning them through `ACTIVE → DECAYED → ARCHIVED → DELETED`.
Emits one lifecycle event per transition. Runs as a background sweep.

```python
class DecayService:
    """
    Implements DecayUseCase.

    Sweep logic (per tenant):
    1. Load RetentionPolicy via TenantPolicyRepositoryPort.
    2. Fetch ACTIVE records older than decay_after_days:
       - Use record_repo.list_by_scope with lifecycle_states=[ACTIVE]
         and a computed cutoff date.
       - Also check sector-specific decay overrides:
         policy.sector_decay_overrides = {MemorySector.EPISODIC: 30, ...}
         Sector override takes precedence over global decay_after_days.
    3. For each qualifying record:
       a. Call record_repo.update_lifecycle(state=DECAYED).
       b. Append AuditEntry (DECAY, SUCCESS).
       c. Emit MemoryDecayedEvent.
    4. Fetch DECAYED records older than archive_after_days (if set):
       - Transition to ARCHIVED.
       - Emit MemoryArchivedEvent.
    5. Fetch ARCHIVED records older than delete_after_days (if set):
       - Transition to DELETED.
       - Emit MemoryDeletedEvent.
    6. Return total records transitioned across all steps.

    Rules:
    - If a threshold is None in policy, skip that transition entirely.
    - Thresholds are measured from record.recorded_at.
    - All transitions must be idempotent: a record already in the target
      state must be silently skipped (no re-emit, no re-audit).
    - process_limit: optional int cap on records processed per run (default 500).
      Prevents unbounded sweeps.
    """

    def __init__(
        self,
        record_repo: MemoryRecordRepositoryPort,
        audit_log: AuditLogPort,
        observer: ObserverPort,
        policy_repo: TenantPolicyRepositoryPort,
        process_limit: int = 500,
    ) -> None: ...

    async def execute(self, tenant_id: TenantId) -> int: ...

    async def _transition(
        self,
        record: MemoryRecord,
        target_state: LifecycleState,
        tenant_id: TenantId,
    ) -> None: ...

    def _effective_decay_days(
        self, record: MemoryRecord, policy: RetentionPolicy
    ) -> Optional[int]: ...
    # Returns sector override if present, else global decay_after_days
```
