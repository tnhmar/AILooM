## Objective

Implement `QdrantVectorIndex` — an alternate `VectorIndexPort` backed by
Qdrant Cloud (or self-hosted Qdrant). Uses collection-per-tenant isolation.
This is the recommended path for teams that want a dedicated vector database
rather than pgvector colocated with records.

**Generate `src/memory_layer/adapters/qdrant/vector_index.py`:**

```python
class QdrantVectorIndex:
    """
    Implements VectorIndexPort.

    Collection naming: f"memory_layer_{tenant_id}".
    Collection is created on first upsert if it does not exist.

    upsert(doc):
      - Point ID = memory_id (deterministic string → UUID5 from memory_id).
      - Vector = doc.embedding.
      - Payload = {
          "tenant_id": ..., "sector": ..., "lifecycle_state": ...,
          "content": ..., "metadata": ...
        }.
      - Use qdrant_client.upsert (not upload_collection).

    search(query_embedding, tenant_id, k, filters):
      - Collection = f"memory_layer_{tenant_id}".
      - Build qdrant Filter from filters dict:
          sectors → FieldCondition(key="sector", match=MatchAny(any=[...]))
          lifecycle_states → FieldCondition(key="lifecycle_state", match=MatchAny(...))
      - client.search(collection_name, query_vector, query_filter, limit=k, with_payload=True).
      - Map ScoredPoint → VectorSearchResult(memory_id, score, content, metadata).

    delete(memory_id, tenant_id):
      - client.delete(collection_name, points_selector=[uuid5(memory_id)]).

    _ensure_collection(tenant_id, dimensions):
      - Create collection if not exists with:
          VectorParams(size=dimensions, distance=Distance.COSINE).
          hnsw_config=HnswConfigDiff(m=16, ef_construct=64).
    """

    def __init__(
        self,
        url: str,
        api_key: Optional[str] = None,
        dimensions: int = 1536,
    ) -> None: ...

    async def upsert(self, doc: VectorDocument) -> None: ...
    async def search(
        self,
        query_embedding: List[float],
        tenant_id: TenantId,
        k: int,
        filters: dict,
    ) -> List[VectorSearchResult]: ...
    async def delete(self, memory_id: MemoryId, tenant_id: TenantId) -> None: ...
    async def _ensure_collection(
        self, tenant_id: TenantId, dimensions: int
    ) -> None: ...
    async def aclose(self) -> None: ...
```

Rules:
- Use `uuid.uuid5(uuid.NAMESPACE_DNS, memory_id)` for stable Qdrant point IDs.
- `_ensure_collection` must be idempotent: use `recreate_collection=False` or
  check existence first.
- Log collection creation at INFO level.
- `aclose()` calls `client.close()`.
