## Objective

Implement structured JSON logging and optional OpenTelemetry distributed tracing.
Every log record must carry `tenant_id`, `trace_id`, and `component` fields.
Tracing instruments the engine use-case boundaries as span roots.
Both features are toggled via `ObservabilitySettings`.
