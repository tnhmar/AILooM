## Objective

Implement `RetrievalService` — executes a `QueryPlan` by running all index queries
in parallel with `asyncio.gather`, then fuses results using Reciprocal Rank Fusion (RRF),
and optionally applies LLM reranking. Returns a ranked list of `SearchResultItem`.
