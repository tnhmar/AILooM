**File 2: `src/memory_layer/domain/exceptions.py`**

```python
class MemoryLayerError(Exception): ...          # base

class TenantIsolationViolation(MemoryLayerError):
    def __init__(self, actor: str, requested_tenant_id: str):
        self.actor = actor
        self.requested_tenant_id = requested_tenant_id
        super().__init__(f"Tenant isolation violation: actor={actor}, requested={requested_tenant_id}")

class MemoryNotFoundError(MemoryLayerError):
    def __init__(self, memory_id: str):
        self.memory_id = memory_id
        super().__init__(f"Memory not found: {memory_id}")

class FactNotFoundError(MemoryLayerError): ...
class IdempotencyConflictError(MemoryLayerError): ...

class CapabilityNotAvailableError(MemoryLayerError):
    def __init__(self, capability: str):
        self.capability = capability
        super().__init__(f"Capability not available: {capability}")

class ExtractionError(MemoryLayerError): ...
class StorageError(MemoryLayerError): ...

class SchemaVersionError(MemoryLayerError):
    def __init__(self, expected: int, found: int):
        self.expected = expected
        self.found = found
        super().__init__(f"Schema version mismatch: expected={expected}, found={found}")

class PolicyError(MemoryLayerError): ...
```

**File 3: `tests/domain/test_types.py`**

Tests to include:

5. `TenantIsolationViolation("alice", "t-99")` has `.actor` and `.requested_tenant_id`.
6. `CapabilityNotAvailableError("graph").capability == "graph"`.
7. `SchemaVersionError(1, 2)` has `.expected == 1` and `.found == 2`.
8. `MemoryNotFoundError("abc").memory_id == "abc"`.
9. All exceptions are subclasses of `MemoryLayerError`.
10. All enums support `str(member)` returning the value string.
