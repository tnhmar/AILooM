## Objective

Implement the four core domain objects: `MemoryRecord`, `Fact`, `AuditEntry`, and `MemoryTrace`.
These are the central data structures of the entire system.
The **critical invariant** for `MemoryRecord` is that `lifecycle_state` defaults to `ACTIVE`
on creation — records are searchable immediately after a durable write (SDS v1.1, Section 7.1).

**Add to `src/memory_layer/domain/records.py`:**

```python
# 1. MemoryRecord — frozen dataclass
# CRITICAL INVARIANT: lifecycle_state defaults to ACTIVE.
# Records are searchable immediately on durable write.
# pipeline_status alone reflects enrichment progress — never lifecycle_state.
@dataclass(frozen=True)
class MemoryRecord:
    id: MemoryId
    tenant_id: TenantId
    scope: Scope
    raw_payload: str
    payload_type: PayloadType
    sector: MemorySector
    lifecycle_state: LifecycleState = LifecycleState.ACTIVE
    pipeline_status: PipelineStatus = PipelineStatus.PENDING
    recorded_at: datetime = field(default_factory=datetime.utcnow)
    idempotency_key: Optional[str] = None
    metadata: dict = field(default_factory=dict)

# 2. Fact — frozen dataclass
@dataclass(frozen=True)
class Fact:
    id: FactId
    memory_record_id: MemoryId
    tenant_id: TenantId
    scope: Scope
    subject_entity_id: EntityId
    predicate: str
    predicate_group: str
    object_value: str
    effective_from: datetime
    effective_to: Optional[datetime] = None
    recorded_at: datetime = field(default_factory=datetime.utcnow)
    supersedes: Optional[FactId] = None
    confidence: float = 1.0
    sector: MemorySector = MemorySector.SEMANTIC
    lifecycle_state: LifecycleState = LifecycleState.ACTIVE

# 3. AuditEntry — frozen dataclass
@dataclass(frozen=True)
class AuditEntry:
    id: AuditId
    tenant_id: TenantId
    scope: Scope
    operation: AuditOperation
    memory_id: Optional[MemoryId] = None
    actor: str = "system"
    timestamp: datetime = field(default_factory=datetime.utcnow)
    outcome: AuditOutcome = AuditOutcome.SUCCESS
    detail: dict = field(default_factory=dict)

# 4. MemoryTrace — regular (mutable) dataclass
@dataclass
class MemoryTrace:
    trace_id: TraceId
    memory_id: MemoryId
    scope: Scope
    write_event: AuditEntry
    enrichment_status: PipelineStatus
    facts_derived: List[FactId] = field(default_factory=list)
    entities_extracted: List[EntityId] = field(default_factory=list)
    mutations: List[AuditEntry] = field(default_factory=list)
    recall_event: Optional[AuditEntry] = None
    recall_signals: Optional[dict] = None
    recall_explanation: Optional[str] = None
    query_plan: Optional[Any] = None
    constructed_at: datetime = field(default_factory=datetime.utcnow)
```
